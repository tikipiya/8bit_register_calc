# 8-Bit Register Simulator (CLI)

C言語で作成された、CPUの内部動作（レジスタ・演算・フラグ）をシミュレートする学習用CLIツール

## 概要
このツールは、GUIではなくあえてCLI上で「CPUがどうやって計算しているか」を視覚化する
ユーザーが簡易的なアセンブリ言語風の命令を入力すると、内部の8ビットレジスタの状態がどのように変化するかをリアルタイムで表示する



## 主な機能
- **8ビット・レジスタ・エミュレーション**: Register A, Register B の2つを搭載。
- **演算フラグ**: 加算時に桁あふれが発生した際の「キャリーフラグ」を再現。
- **バイナリ・ビジュアライザ**: レジスタの中身を10進数だけでなく、ビット単位（0101...）で表示。
- **カラー出力**: エスケープシーケンスを利用し、ビットの「1」や重要な変化をカラーで強調。

## 学べる概念
- **ビット演算**: 数値が内部でどのように2進数として扱われているか。
- **命令解析（パース）**: 文字列入力を読み取り、実行可能な処理に変換する仕組み。
- **条件分岐と構造体**: CPUの状態をデータ構造として管理する方法。

## 使い方

### 1. コンパイル
標準的なCコンパイラ（gccなど）を使用してコンパイルします。
